# High Tech Academy Landing Page

Одностраничный сайт High Tech Academy на обычных `HTML`, `CSS` и `JavaScript`.

## Где Что Менять

### `script.js`

Главный файл для редактирования контента. В верхней части файла находятся данные, которые чаще всего нужно менять:

- `IMAGE_DIRS` — папки с изображениями.
- `COMPANY_LOGO_*` — логотипы компаний в блоке компаний.
- `PARTNER_LOGOS` — логотипы партнёров в курсе предпринимательства.
- `PHOTO_HERO` — главное фото первого экрана.
- `PHOTO_ACHIEVEMENTS` — фото достижений.
- `PHOTO_VIDEO_THUMBS` — обложки видео в блоке «Голоса HTA».
- `PHOTO_LIFE` — фото бегущей галереи «Жизнь в HTA».
- `pageData` — карточки, проекты, стартапы, видео и дата дня открытых дверей.
- `MBACC_VIDEOS` — ссылки на видео в блоке Minerva/MBacc.

Обычные правки текста, ссылок, фото и логотипов лучше делать именно в `script.js`, не трогая функции ниже.

### `index.html`

Структура страницы. В файле расставлены комментарии перед главными секциями:

- `HEADER`
- `HERO`
- `WHY HTA`
- `UNIQUE FEATURES`
- `MBACC`
- `LIFE HTA`
- `ACHIEVEMENTS`
- `ENTREPRENEURSHIP`
- `TESTIMONIALS`
- `PD`
- `OPEN DAY`
- `FOOTER`
- модальные окна

Меняйте `index.html`, если нужно добавить, убрать или переставить целую секцию.

### `styles.css`

Внешний вид сайта. В начале файла есть карта CSS. Основные зоны:

- `Design tokens` — цвета, тени, скругления.
- `Hero` — первый экран.
- `Entrepreneurship course` — курс предпринимательства.
- `MBacc block` — блок Minerva/MBacc.
- `Testimonials video` — карусель видео «Голоса HTA».
- `PD` — блок профессионального развития.
- `Modals` — всплывающие окна.
- `Responsive` — адаптив под планшеты и телефоны.

## Частые Задачи

### Поменять Фото

1. Положите новый файл в нужную папку внутри `images/`.
2. Откройте `script.js`.
3. Найдите нужный массив `PHOTO_*`.
4. Замените имя файла.

### Поменять Логотип Партнёра

1. Положите логотип в папку `logo/`.
2. Откройте `script.js`.
3. Найдите `PARTNER_LOGOS`.
4. Пропишите путь, например:

```js
uvu: "./logo/logo-uvu.png",
```

### Добавить Ссылку К Проекту

В `script.js` найдите `pageData.entrepreneurship`, нужный год и нужный проект. Добавьте ссылку в `links`:

```js
links: [
  { label: "Видео 1", url: "https://..." },
  { label: "Статья", url: "https://..." },
],
```

Если `links: []`, на сайте появится текст «Ссылка скоро».

### Изменить Видео Minerva/MBacc

В `script.js` найдите `MBACC_VIDEOS` и замените `embedUrl`.

Обложки этих видео меняются в `styles.css` в блоке `MBacc block`.

### Изменить Дату Дня Открытых Дверей

В `script.js` найдите:

```js
openDay: {
  dateLine: "Апрель 2026 — дата уточняется",
}
```

И замените текст.

## Как Запустить

Откройте `index.html` в браузере. Для проверки после правок обновляйте страницу через `Ctrl + F5`, чтобы браузер не показывал старую версию CSS/JS из кэша.
