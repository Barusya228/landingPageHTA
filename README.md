# High Tech Academy — Лендинг приёма в школу

Одностраничный лендинг для школы High Tech Academy на чистом **HTML, CSS и JavaScript**.

## Структура проекта

- `index.html` — разметка страницы с семантическими секциями (hero, о школе, цифры, преимущества, даты, галерея, форма, подвал).
- `styles.css` — стили, тетрис‑образная геометрия, адаптивная сетка, hover‑эффекты и анимации.
- `script.js` — логика анимаций при скролле, счётчики статистики, генерация блока с датами и валидация формы.

## Как запустить

1. Откройте файл `index.html` в браузере (двойной клик или через локальный сервер).
2. При необходимости измените важные даты в массиве `importantDates` в файле `script.js`.
